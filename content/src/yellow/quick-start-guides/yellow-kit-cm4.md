## Prerequisites

- Home Assistant Yellow
- Heat sink assembly bag
- Ethernet cable
- Power supply

Additionally, make sure you have the following items which are not part of the Kit

- Raspberry Pi Compute Module 4 (CM4)
- USB flash drive
